Процесс загрузки информации

1) load_sitemap.php - загружает новые страницы с коннекторов со статусом NEW
1.1) проверяем лоадеры, при необходимости делаем неактивными
######################################################
2) create_cache.php - загружаем кэш страниц и изображений
2.0) load_data.php - загрузка данных на предмет дублей(отдельная обработка кэша и извлечение данных). по требованию
2.1) мапим новые характеристики
2.2) мапим категорий
2.3) мапим производителей
######################################################
3) create_pproduct.php - создает pproduct, и прописывает его принадлежность в лоадере
######################################################
4) create_product.php - создает товар и сразу загружает всю информацию по лоадерам
######################################################
5) update_product.php
обновляет товары по выборке. из кэша или сразу с сайта создавая кэш
5.1) update_product_features.php
обновляет характеристики товара диапазоном.
